# React, Typescript and redux-saga's example application

## Features
- * [x] ReactJS 
- * [x] Typescript 
- * [x] React router
- * [x] Redux 
- * [x] Immutable 
- * [x] Redux-saga

## Getting started
1. install module
```bash
# install modules
$ npm install
```

2. start frontend server
```bash
$ npm start
```

3. start deployd module
```bash
# If it is a first time to set this application.
$ npm run dpd:create

$ npm run dpd:up
```
